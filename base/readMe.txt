Usage:
	Install all dependencies using the following command:
		npm install

	To start the server use the following command:
		node server.js

	To connect to the server navigate to the following URL using a browser:
		http://localhost:2406/




Development Environment:
	Operating System:
		Windows 10 64 bit

	Browser Compatibility:
		+ Chrome (primary)
		+ Firefox
		+ Edge
		+ Internet Explorer
